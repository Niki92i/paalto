# Support

## Where to Ask

- GitHub Discussions: design questions, usage help, integration patterns.
- GitHub Issues: reproducible bugs, docs gaps, broken examples.
- Pull requests: new skills, examples, docs, validation improvements.

## Before Filing

Run:

```bash
node scripts/validate-skills.mjs
```

Then include:

- Your host environment (Claude Code version, OS, Node version).
- Which workflow or skill you ran.
- The relevant non-sensitive excerpt from `runs/<run_id>/events.jsonl`.
- The integration involved, if any.

Never paste API keys, private customer transcripts, or unsanitized run folders.
